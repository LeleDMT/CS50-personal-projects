How did this website works?

This website uses machine learning, a subset of artificial intelligence (AI), that uses data to train model that will predict certain outcomes

For this project, I used a dataset of the Big Five from Kaggle, composed of 1 million answer of the Big Five test from different user around the world.

From this dataset, I trained a machine learning model with 25 000 answers (because 1 million would have been way too much for my computer) using a clustering algorithm called KMeans.

This model allowed me to find pattern of personnality that emerged from the data, and regroup them in different clusters.
